# Bangalore House Price Predictor

A Machine Learning based web application that predicts Bangalore house prices based on user inputs such as location, total square feet, number of bathrooms and BHK.

## Project Overview

This project uses Machine Learning regression techniques to predict house prices in Bangalore. A complete ML pipeline is created including data preprocessing, feature engineering, model training and deployment using Streamlit.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
- Streamlit

## Machine Learning Workflow

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Preprocessing
- Model Training
- Model Evaluation
- Deployment using Streamlit

## Features Used

- Location
- Total Square Feet
- Number of Bathrooms
- BHK (Bedrooms)

## Model Used

- Ridge Regression

## Model Pipeline

The model uses:

- OneHotEncoder for categorical feature (Location)
- StandardScaler for feature scaling
- Ridge Regression for prediction
